Using This Device

This Pi Zero can:

  • Emulate a USB keyboard
  • Broadcast Wi-Fi for remote control
  • Send predefined or custom keystrokes to a target

Use it to test USB-based injection attacks on Windows/macOS/Linux systems.

Input Script Format

The device parses and executes a series of keystrokes written in plain text. Each line is interpreted either as a keystroke command or a delay.

Supported Input Examples:

# Send lowercase text
hello world

# Send uppercase letters and special characters
Hello, World!

# Simulate holding modifier keys
CTRL+ALT+DEL
SHIFT+A

# Add a delay in seconds
DELAY 1.5

# Combine multiple commands
GUI+r
DELAY 0.5
cmd
ENTER

Special Keys Supported:

  • ENTER, TAB, ESC, BACKSPACE
  • Modifier keys: CTRL, SHIFT, ALT, GUI (Windows/Command key)
  • Function keys: F1 to F12
  • Symbols and characters (e.g., !@#$%^&*)

Tips:

  • Each line is processed top to bottom.
  • Combos like CTRL+C or GUI+R send simultaneous key presses.
  • Use DELAY X between commands to give the target system time to respond.
